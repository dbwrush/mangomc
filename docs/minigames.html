Minigames

MangoMC is always looking to improve our selection in minigames. Currently we have 2 custom minigames created by sudologic.

ElytraDogfights

An aerial combat minigame created by sudologic, players spawn into a large spherical arena and dogfight to be the last individual or team flying. Hold ‘Q’ to rapid-fire, use the slime blocks to get a boost, and avoid getting caught in the cobwebs! If you want to add this plugin on your own server you can download it from spigotmc.org

Ledges

Loosely inspired by Super Smash Bros, Ledges is a game about knockback. Each hit you take increases the amount of knockback you will take until you fall off the ledges. View your stats by running the /showleaderboard command. If you want to add this plugin on your own server you can download it from spigotmc.org
